Constant Volume + Energy

Set: box size only
🌡️ NVT — Temperature is controlled, volume is fixed. You will set start/end temperature and box dimensions.
🔩 NPT — Both temperature and pressure controlled. Box volume adjusts. You will set temperature, pressure, and initial box dimensions.
⚡ NVE — No thermostat or barostat. Total energy is conserved. Typically used after equilibration. Only box size required.

🖥 Open Full Trajectory in VMD
Download system.data and traj.lammpstrj below, then in VMD Extensions → Tk Console, paste: 
package require topotools
topo readlammpsdata "system.data" full
mol addfile "traj.lammpstrj" type lammpstrj waitfor all
mol delrep 0 top
mol representation VDW 1.0 12.0
mol color Name
mol selection all
mol material Opaque
mol addrep top
display projection Orthographic
animate goto start
